Installation

Requirements

  1. DGA requires Java 2 Standard Edition (J2SE) 5.0.

  2. Minimum hardware requirements for running DGA are a CPU of at least 133 MHz and 64M of RAM memory.

Installation on Windows

  1. Install Java 2 on your system. If Java 2 is already installed, skip this step. Dowload the installation kit for your platform from java.sun.com. Either the J2SE development kit (JDK) or just the J2SE Runtime Environment (JRE) runtime environment can be used. JRE 5.0 or later is required.
  2. Run the installer dga.msi.

Installation on other OS

  1. Install Java 2 on your system. If Java 2 is already installed, skip this step. Dowload the installation kit for your platform from java.sun.com. Either the J2SE development kit (JDK) or just the J2SE Runtime Environment (JRE) runtime environment can be used. JRE 5.0 or later is required.
  2. Make sure that the PATH variable contains the path to the java executable. You can check installation of the proper version of Java by issuing the command java -version. You should get an output like this: 
    
    > java -version
    java version "1.5.0_06"
    Java(TM) 2 Runtime Environment, Standard Edition (build 1.5.0_06-b05)
    Java HotSpot(TM) Client VM (build 1.5.0_06-b05, mixed mode)

    >

  3. Uncompress the archive dga.tgz: 

    
    tar zxf dga.tgz
    This will create a directory DGAnnotator.

  4. In order to run DGA, issue the following command in the directory DGAnnotator: 

    
    java -jar dga.jar
