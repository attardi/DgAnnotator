Annotation

The annotation process starts by selecting a sentence to be annotated from the displayed text in the lower panel.

Text can be entered directly in the tab named Scratch: a sentence can be selected from this tab by dragging the mouse while pressing the left button. Once a fragment of text (sentence) is selected, the tool bar button is enabled. Pressing this button, the selected text is submitted to the POS tagger for tagging and then it is displayed in the annotation panel.

Sentences loaded from corpus files can just be selected with a mouse click.

POS Tagging

Select the POS tag for a word from the line below the sentence: the list of available POS tags for the given corpus will appear in the pane to the left; select (left mouse button) the appropriate one to annotate the word.

Dependency Tagging

In order to create a dependency relation between two words, first select (left mouse button) the dependent word (the one from where the arc starts). This word gets marked in blue.

Next, click on the head word: an arc from the dependent word to the head word will be created. This arc will be labeled with an empty label "[ ]".

In order to assign a dependency relation to an arc one must select its label and then click on the appropriate label in the left panel.

A dependency relation can be deleted by selecting its arc label and pressing the Delete key.

According to the dependency grammar formalism, each token in a sentence (word or punctuation) must depend on a single other one. The only exception is the head of the whole sentence (usually the main verb), which depends on no other word. Specifying that a word is the head of a sentence is achieved as follows: click on the head word to mark it, then click on the word itself for indicating it as head of the sentence. Assigning a type to this new arc (typically ROOT) is performed like for others relation types.

Since a sentence can have just one head word, DGA allows defining only one such word. Once a head word for a sentence has been identified, no other word can be marked as head word.

The actions for creating and labeling dependency relations can be performed repeatedly until the user considers the sentence to be properly and completely annotated.

An annotated sentence can be saved by using the menu command Append To.., which lets the user choose a file to which the sentence will be added.

If another sentence is selected for being annotated witthout having being saved, the user is notified and is allowed to save the sentence, even if it is not completely annotated.
