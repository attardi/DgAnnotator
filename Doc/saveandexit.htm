Saving data and exiting

The annotated sentences are saved to a file only when the user explicitly chooses either the Save or the Save as command from the File menu. The annotations will be saved either in the DGA XML format, or in the CoNLL tab format, depending on whether the file extension is .xml, .conll or .tab.

The Append To ... command appends the annotated sentences to an existing file.

Ending a working session takes place by selecting the Exit command from the File menu, or by closing the main window of the application. If, when exiting, some annotated sentences have not been saved, the user is notified and has the opportunity to save them.
