Viewing and modifying an annotated text

DGA allows viewing and possibly modifying the annotations of previously annotated texts. Through the Open command from menu File one can select the file containing the annotated text. The selected file must be either:

  • an XML file, having a structure in accordance with the DGA format (see The XML format used by DGA)
  • a file in the CoNLL tab format, with extension .conll or .tab
    • .

The content of the chosen file will be displayed within a new window tab, one sentence per line. When clicking on a sentence, its annotation will be displayed in the annotation panel. Annotations can be edited here (see the section referring to annotation).

Saving modifications is performed by means of the commands Save or Save as from menu File.
